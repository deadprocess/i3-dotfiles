# i3-dotfiles

My personal i3 dotfiles for Arch Linux.

## Setup

Run from inside the repo directory:

```bash
bash setup.sh
```

## After install

1. Add wallpapers to `~/wallpaper/`
2. Generate color theme: `wal -i ~/wallpaper/`
3. Start i3: `startx`

## What's included

- i3 — window manager
- i3status — status bar
- kitty — terminal
- rofi — app launcher
- fastfetch — system info
- fcitx5 — input method
